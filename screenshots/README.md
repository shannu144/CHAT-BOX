# Screenshots Directory

This folder contains screenshots of VITCHAT for the README.

## Required Screenshots:
1. `login.png` - Login/Signup page
2. `chat.png` - Main chat interface
3. `ai-chat.png` - AI bot conversation
4. `emoji-picker.png` - Emoji picker in action
5. `voice-notes.png` - Voice note recording
6. `online-status.png` - Online status indicators
7. `settings.png` - Settings modal
8. `banner.png` - App banner/logo

## How to Add Screenshots:
1. Open VITCHAT in your browser
2. Take screenshots using your OS screenshot tool
3. Save them in this folder with the names above
4. Commit and push to GitHub
